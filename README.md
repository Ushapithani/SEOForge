# SEOForge 🚀

SEOForge is an AI-powered SEO auditing platform that helps website owners, developers, and digital marketers analyze, optimize, and improve their search engine rankings.

🌐 **Live Demo:** https://rank-pilot-omega.vercel.app/

With instant SEO audits, performance analysis, keyword insights, and actionable recommendations, SEOForge makes it easy to identify issues and discover opportunities to boost organic traffic.

## ✨ Features

* 🔍 Instant SEO Audits
* ⚡ Performance Analysis
* 🎯 Keyword Insights
* 🤖 AI-Powered Recommendations
* 📊 Detailed Reports

## 🚀 Installation

```bash
git clone https://github.com/Ushapithani/SEOForge.git
cd SEOForge
npm install
npm run dev
```
