# SEOForge 🚀

SEOForge is an AI-powered SEO auditing platform that helps website owners, developers, and digital marketers analyze, optimize, and improve their search engine rankings.

With instant SEO audits, performance analysis, keyword insights, and actionable recommendations, SEOForge makes it easy to identify issues and discover opportunities to boost organic traffic.

## ✨ Features

### 🔍 Instant SEO Audits

* Analyze website SEO health in seconds
* Detect on-page SEO issues
* Review meta tags, headings, and content structure

### ⚡ Performance Analysis

* Evaluate website speed
* Monitor Core Web Vitals
* Improve user experience and search performance

### 🎯 Keyword Insights

* Discover relevant keywords
* Analyze keyword opportunities
* Optimize content for better visibility

### 🤖 AI-Powered Recommendations

* Smart SEO suggestions
* Content quality analysis
* Technical SEO improvements
* Step-by-step optimization guidance

### 📊 Detailed Reports

* SEO health score
* Performance metrics
* Priority-based recommendations
* Easy-to-understand dashboards

## 🛠️ Tech Stack

* React 19
* TypeScript
* React Router
* Tailwind CSS 4
* Vite
* Lucide React
* React Simple Icons

## 🚀 Installation

```bash
git clone https://github.com/Ushapithani/SEOForge.git
cd SEOForge
npm install
npm run dev
```

## 📦 Production Build

```bash
npm run build
npm run preview
```
